# ktds-MS-AI

1. 프로젝트 골 및 주요 가치
   실시간 고객 불만, 건의, 문의 유형 자동 집계·분석
   반복·긴급 패턴 탐지 및 IT 서비스 개선 후보군 자동 추출
   IT 서비스 만족도를 높여주는 근거 기반 인사이트 제공

2. MVP 핵심 기능 설계
   기능명 설명 기대효과
   VOC 데이터 자동 수집 고객센터 콜, 이메일, 채팅 등 VOC 원문 자동 통합 데이터 전처리 효율화
   텍스트 분류/키워드 추출 AI 기반 이슈 유형 분류, 주요 키워드/불만/개선 워딩 자동 추출 주요 불만/이슈 파악·트렌드 분석
   우선순위 자동 평가 불만 빈도, 심각도, IT영향도 등 다면분석으로 개선 필요도 자동 산출 신속한 개선 어젠다 수립
   대시보드/리포트 시각화 인사이트 데이터 시각 제공(월간 주요 VOC, change tracking, 워드클라우드 등) IT-사업부, 경영진 소통 원활
   샘플 개선 방안 제안 유사 사례 추천, 예상 개선 효과(시뮬레이션) 구체적 실행 제안 도출
